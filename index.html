The Basics

Let's first learn how Liquid works! It's a super friendly and easy to learn programming language
Well see how Liquid looks in code. If you see {{ or }} that means you've found an Object.
If you see {% or %} you've found some tags!
So what's the difference between the two of these? Well {{ }} is for printing information. We'll see that done soon!
{% %} is used for logic ie. we're asking the code to preform some kind of action. Examples coming below!

Objects

Let's just jump right in an use a very simple example to get started of an object
{{ product.title }} - We've asked the webpage for basic information and to print it!
It's going to return the Title of the product of the page you're on!
We'll go with Shopify T-Shirt for now!

Tags

We'll then look into a tag! {% assign myName = 'Dale' %} - now I've told myName to become Dale when ever I write it within an Object.
What does that mean?
Simple! When I now write {{ myName }} it will print Dale!

Filters

There are also Filters that can add additional information inside the Object.
Using the above example we can use {{ myName | upcase }} to print DALE instead of Dale.

Working Example

Taking an actual example of where this would all work would be:
{% for product in collections.all.products %}
{{ product.title }}
{{ product.price | money }}
{% endfor %}
This will return all of the products in the collection with the handle of 'all' and only display the product titles & prices for those products.
We'll see how for statements work at a later time!
